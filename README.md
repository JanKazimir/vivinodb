# Vivino Market Analysis
This project is an analysis of a vivino dataset as part of a becode bootcamp exercise targeted at sql pratice.   

You can have a look at the result here:
https://vivinodb-mdjjynuupbu4qgxkj3quxz.streamlit.app/


## Description
The vivino dataset is a partial snapshot focusing on the upper end of the market. The data is healthy, with no missing values. The only issue is that among all the wineries in the database, only four actually are linked to wines for which we have data.
